# CPSC 314

Haskell course quizzes and assignments.

Lecture PDFs are in `lectures/`.

---

## Quizzes

Each quiz is a React app. To run any of them:

```bash
cd quiz2   # replace with quiz3, quiz4, quiz5, quiz6, quiz7, or quiz8
npm install
npm run dev
```

Then open the localhost URL shown in the terminal.
